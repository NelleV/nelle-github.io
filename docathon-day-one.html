Software Engineers positions to work on numpy at BIDS

Posted on Mon 06 March 2017 in info

BIDS recently secured funding for two software engineers to work on numpy. All the contributions will go through the usual contribution workflow of the project via the standard pull request reviews on GitHub. The exact scope of the contributions can be discussed based on the applicant's interests, areas of expertise and relevance with the ongoing efforts. Potential contributions include:

  • Refactoring the dtype system, for easier extension to custom data (e.g, categorical data)
  • Missing data support
  • Duck arrays
  • Migration of C-Python interfacing code to Cython

We are looking for candidates with a proven developer experience in Python and C/Cython, preferably with the scientific Python ecosystem (NumPy, SciPy, ...) and will favor applications with a track record of contributions to scientific Open Source projects such as NumPy, SciPy or related ones.

Both US and international applicants are eligible.

  • Duration: 2 years, starting as soon as possible.
  • Location: Berkeley, at BIDS

Interested applicants can find more more information in the job posting.

If you are interested, please feel free to send me an email privately with Nathaniel and Stéfan in CC.
